### git 的流程
*  下载安装git
*  git 生成 本地 ssh [链接](https://wiki.jikexueyuan.com/project/git-tutorial/remote-repository.html)
* 将本地 ssh 配置到远端服务器
* git init 本地项目
* 将本地项目和远程项目关联,并且推送项目
* git 本地新建分支 推送远端

### git 常用命令行
* 拉代码
```js
  git pull
```
* 推代码
```js
  git push
```
